# 1. Python

  ## 1-A. Variable
  ```
  variable_name = 5                                             // Number
  variable_name = "This is my variable!"                        // String
  ```

  ## 1-B. Constant
  - 대문자로 표기하지만 파이썬에서는 지원되지 않음 
  ```
  PI = 3.14
  ```


# 2. C
  - 선언은 시작 지점에 한다.
  
  ## 2-A. Variable
  ```
  data_type variable_name;                                      // declaration
  variable_name = something;                                    // Assignment
  
  int myInt;
  myInt = 1;
  ```
  
  ## 2-B. Constant
  ```
  const int MY_BIRTHDAY_YEAR = xxxx;
  const float PI = 3.14;
  ```
